# GOAL — 매일 CLE2 개선 질문하기

## 목표
매일 아침 CLE2의 현재 상태를 4가지 패턴 관점에서 점검하고, 랜덤으로 생성된 개선 질문에 대해 생각하고 정리하여 이슈 코멘트로 등록한다.

## 성공 기준 (Definition of Done)
- [ ] 매일 1개 이상의 개선 질문이 CLE2 이슈 코멘트로 등록됨
- [ ] 4가지 패턴(축적 구조, 멀티 관점, 검증/재현성, 프로세스 재설계)이 순환적으로 다뤄짐
- [ ] 누적된 질문/답변이 CLE2 개선 로드맵에 반영됨

## 범위
### 포함 (In Scope)
- CLE2 전체 태스크(CLE2-1 ~ CLE2-8) 대상 일일 개선 질문 생성
- 4가지 패턴 기반 질문 템플릿 및 랜덤 선택 로직
- 이슈 코멘트로 질문+생각 정리 등록

### 제외 (Out of Scope)
- CLE2 외부 프로젝트에 대한 질문
- 질문에 대한 실행 자체 (별도 태스크로 처리)
