# GOAL — Solana 기반 3개 DApp 개발

## 목표
Solana Devnet 위에 출석, NFT 쿠폰, 로컬 크레딧 3개 DApp을 개발하고 GitHub Pages에 배포한다. 기존 ai-solana-agent 코드를 베이스로 활용한다.

## 가치 가설
- 사용자 가치: 블록체인 DApp 개발 역량을 Solana 생태계에서 입증하는 포트폴리오
- 기본 목표: 3개 DApp 핵심 기능 작동 + GitHub Pages 배포 (7/23 제출)
- 도전 목표: QR 인증, 위치 기반 기능까지 구현하여 차별화

- [ ] 7/23 제출 가능 상태
