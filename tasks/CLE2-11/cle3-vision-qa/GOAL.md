# GOAL — CLE3 Vision QA / 결과 검수 시스템

## 목표
생성된 패널 후보를 시각적으로 검수하고 승인, 반려, 수정 요청으로 연결하는 CLE3 Vision QA 기준을 정의한다. 단순 규칙 점검이 아니라 장면 의도와 실제 결과가 맞는지 판단하는 흐름을 정리하는 것이 목적이다.

## 성공 기준 (Definition of Done)
- [ ] `Panel.selectedResult` 선택 기준이 문서화된다
- [ ] ReviewNote의 `clarity`, `composition`, `emotion` 카테고리 연결 규칙이 정리된다
- [ ] 사람 승인 포인트와 AI 재시도 포인트가 구분된다
- [ ] CLE2-9 객체 모델 및 CLE2-10 린트와의 경계가 명확해진다
- [ ] CLE2 대시보드에서 CLE2-11 상태를 추적할 수 있다

## 범위
### 포함 (In Scope)
- 패널 시각 검수 루브릭
- 승인/반려/수정 요청 상태 모델
- 장면 의도 대비 결과 해석 기준
- ReviewNote 기반 검수 기록 방식

### 제외 (Out of Scope)
- 이미지 생성 모델 자체 개선
- 정합성 린트 규칙 엔진 구현
- 배포 후 독자 반응 분석 전체 자동화
