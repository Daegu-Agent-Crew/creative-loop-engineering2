# GOAL — CLE3 Episode Workspace / 에피소드 작업 뷰

## 목표
Story, Episode, Scene, Panel, Results 객체를 실제 작업 흐름으로 묶어 보여주는 에피소드 작업 뷰 기준을 정의한다. `three-body-comic`의 script / storyboard / panels / release / results 구조와 연결되는 작업 화면이 목표다.

## 성공 기준 (Definition of Done)
- [ ] Episode 중심 작업 뷰의 핵심 화면과 정보 구조가 문서화된다
- [ ] script / storyboard / panels / release / results 연결 방식이 정리된다
- [ ] CLE2-7 STUDIO와의 경계가 명시된다
- [ ] CLE2-9 객체 모델과의 관계가 명시된다
- [ ] CLE2 대시보드에서 CLE2-12 상태를 추적할 수 있다

## 범위
### 포함 (In Scope)
- 에피소드 중심 작업 뷰 정보 구조
- 객체 간 연결 방식 시각화
- STUDIO 운영 보드와 창작 객체 뷰의 역할 분리

### 제외 (Out of Scope)
- 실제 완성 UI 전면 구현
- 이미지 생성/배포 자동화 백엔드 구축
- 공개 뷰어 전체 대체
