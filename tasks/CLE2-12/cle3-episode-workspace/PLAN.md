# PLAN — CLE3 Episode Workspace / 에피소드 작업 뷰

## 실행 계획

### Phase 1: 작업 뷰 목적과 핵심 객체 연결 정의
- **담당**: 대구루
- **입력**: `OBJECT-MODEL.md`, `FOLLOW-UP-ISSUES.md`
- **출력**: 정보 구조 초안
- **예상 시간**: 0.5일

### Phase 2: `three-body-comic` 실제 구조와 매핑
- **담당**: 대구루
- **입력**: Phase 1 산출물
- **출력**: script/storyboard/panels/results 연결표
- **예상 시간**: 0.5일

### Phase 3: STUDIO와의 경계 및 CLE2 UI 반영 기준 정리
- **담당**: 대구루
- **입력**: Phase 2 산출물
- **출력**: 화면 역할 분리 문서
- **예상 시간**: 0.5일
