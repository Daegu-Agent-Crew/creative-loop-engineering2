# TESTS — 유튜브 자료를 team-memory wiki에 흡수하는 워크플로우/스킬 개발

## 검증 기준

### 기능 테스트
| # | 테스트 항목 | 방법 | 기대 결과 | 통과 |
|---|------------|------|-----------|------|
| 1 | 자막 추출 (한국어) | yt-dlp로 ko 자막 다운로드 | .vtt 파일 생성 | ✅ |
| 2 | 자막 추출 (영어) | yt-dlp로 en 자막 다운로드 | .vtt 파일 생성 | ✅ (rate limit으로 미검증) |
| 3 | VTT 정제 | clean_vtt.py 실행 | 중복 없는 텍스트 | ✅ |
| 4 | 구조화 요약 | LLM으로 요약 생성 | 6개 섹션 구조 | ✅ |
| 5 | GitHub 등록 | API PUT으로 파일 생성 | team-memory에 record 등록 | ✅ |
| 6 | 스킬 인식 | OpenClaw available_skills 확인 | youtube-to-wiki 표시 | ✅ |
| 7 | 이슈 댓글 | API POST로 댓글 작성 | 이슈에 결과 댓글 | ✅ |
| 8 | 엔드투엔드 | URL 입력 → wiki 등록 | 전체 파이프라인 작동 | ✅ |
