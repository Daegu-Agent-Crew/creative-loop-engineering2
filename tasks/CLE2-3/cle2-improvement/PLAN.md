# PLAN — CLE2 개선: AI 에이전트 병렬 작업 지원 체계 구축

## 실행 계획

### Phase 1: 디렉토리 구조 + 템플릿
- **담당**: 대구루
- **입력**: 이슈 #4 요구사항
- **출력**: tasks/README.md, tasks/_template/{GOAL,PLAN,STATUS,TESTS}.md
- **예상 시간**: 1시간
- **상태**: ✅ 완료

### Phase 2: 기존 이슈 문서 작성
- **담당**: 대구루
- **입력**: Phase 1 템플릿, 기존 이슈 내용
- **출력**: tasks/CLE2-1, CLE2-2, CLE2-3 각각 4개 문서
- **예상 시간**: 1시간
- **상태**: ✅ 완료

### Phase 3: 웹앱 태스크 대시보드
- **담당**: 대구루
- **입력**: Phase 1-2 산출물, 기존 app.js/styles.css
- **출력**: Tasks 라우트, 태스크 카드 UI, 진행률 시각화, 탭 뷰
- **예상 시간**: 3시간
- **상태**: 🔄 진행 중

### Phase 4: 테스트 및 배포
- **담당**: 대구루
- **입력**: Phase 3 완성본
- **출력**: 로컬 테스트 통과, PR 생성, GitHub Pages 배포
- **예상 시간**: 1시간
- **상태**: ⏸️ 대기

## 의존성
- 기존 CLE2 코드베이스 (app.js, styles.css, index.html)

## 리스크
- app.js 파일이 이미 1000줄 이상 → 별도 tasks.js로 분리 또는 인라인
- 정적 파일 기반이므로 실시간 갱신 불가 → 데이터를 app.js 내장으로 처리
